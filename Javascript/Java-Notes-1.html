JAVASCRIPT NOTES

Javascript 8 data types

  • Number

    • Basic numbers used for operations like adding/subtracting. Also includes special values such as Infinity, -Infinity, and NaN (not a number)

  • BigInt

    • For extremely large integers, elaborated later.

  • Strings

    • Strings are words used as variables that can be spliced together or replaced, they must be surrounded by one of 3 different types of quotes.

    • Double Quotes ( "Beep" )
    • Single Quotes ( 'Beep' )
    • Backtick Quotes ( `Beep` )

    The difference between these quotes is that Double and Single are both "simple" and functionally work the same in javascript.

    Backtick quotes are "extended functionality" and allow embedding of variables and expressions into the string by wrapping them in ${. . .}

    Ex:
    let name = "Legendary FSM";
    alert( `Hello, ${name}!`);

    This would result in the console giving [ Hello, Legendary FSM! ]

    Please note that doing the same with single or double quotes would give [ Hello, ${name}! ]

  • Symbol

    • Used to create unique identifiers for Objects.

  • Object

    • Objects are special, as they are the only non "primitive" data type. This means that they are able to store collections of data and more complex entities, rather than simple values such as strings or numbers.

  • Boolean

    • True or False, and as a result from comparisons.

  • Null

    • "null" refers specifically to nothing, or an empty value.

  • Undefined

    • Refers to a variable that is declared, but is not assigned a value.

The typeof operator

The typeof operator will return a string with the type of data the given variable is.

Example: typeof 0 // returns "number"
